####		Adaming EtablissementScolaire		####

L'application possède 6 controllers, un par entité.
Tous retournent une Responseentity contenant : 
	- Un boolean isError false par default, true si la requète a engendré une erreur.
	- Un String message
	- Un objet body, contenant un objet si besoin, null sinon.
Tous les controller implémentent une interface :
	-methode post : create
	-methode delete : deleteById
	-methode put : updateById
	-mmethode get : /one readById
			/all readAll
